# Color Flipper Web App

This project is a simple web application that allows users to change the background color of a webpage. The user can choose between fixed colors or generate a random color.

## Features
- Change background color to Green, Red, or Blue
- Generate a random color using RGB values
- Instant visual update when a button is clicked

## Technologies Used
- HTML
- JavaScript
